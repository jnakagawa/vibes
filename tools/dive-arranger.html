Dive Arranger beta

A Chrome extension — drag the tiles of any MotherDuck dive, then Submit to write the new layout back.
what it does

MotherDuck dives are React components; their charts render as tiles inside a sandboxed iframe. Dive Arranger draws drag/resize handles on the real rendered tiles and lets you rearrange them WYSIWYG into a 12-column grid — reorder, put tiles side by side, span, set heights, or hit ✕ to delete a tile (an empty spacer div or a whole chart; pinned conditional/loop blocks are protected).

The only write path is an explicit Submit: it regenerates the dive's source from your layout, writes it back via MotherDuck, and verifies the read-back. Nothing is mutated until you click it.

install (load unpacked)
  1. Grab the code — it's in this repo under tools/dive-arranger/. The build output (dist/) is committed, so no build step is required. (To rebuild: npm install && npm run build in that folder.)
  2. Open chrome://extensions in Chrome.
  3. Turn on Developer mode (top-right toggle).
  4. Click Load unpacked.
  5. Select the tools/dive-arranger/ folder (the one with manifest.json).
  6. Open any MotherDuck dive, then click the ⠿ Arrange button at the bottom-right. Drag tiles by the grip, drag their right/bottom edges to resize, and hit Submit when it looks right.

Auth: the extension picks up a short-lived MotherDuck token from the dive page automatically; if none is found, paste one on the extension's Options page. The token never leaves your browser.
