# SF Top Restaurants — Google Maps Saved List

A curated list of San Francisco's top restaurants, all rated **7.5/10 or higher**
(Google rating × 2), each pin carrying its rating and a short blurb.

## 📍 Get it into Google Maps (one-time, ~30 seconds)

Google doesn't allow apps to write to your personal "Saved" lists, so the list is
delivered as a **Google My Maps** import — which shows up in the Google Maps app
under **Saved → Maps** and opens as pins like any saved list:

1. Open [Google My Maps](https://www.google.com/mymaps) → **Create a new map**
2. In the layer panel click **Import** and upload
   [`sf-top-restaurants.csv`](https://raw.githubusercontent.com/jnakagawa/vibes/claude/maps-restaurant-player-qm4ypw/maps/sf-top-restaurants.csv)
   (download it first, or paste the URL where supported)
3. When asked which column places the pins → choose **Address**
4. When asked which column titles the pins → choose **Name**
5. Name the map "SF Top Restaurants" → **Share** → grab the link

That share link now opens the full pinned map in Google Maps on any device, and
it appears in your Google Maps app under **Saved → Maps**. Tap any pin to see
the rating and blurb.

## 🔄 Staying updated

This CSV is refreshed on a schedule (ratings re-checked, anything that slips
below 7.5 dropped, notable newcomers added). Google doesn't let My Maps
auto-sync from a URL, so to pull the latest: open your map in My Maps →
delete the layer → re-import the CSV from the link above.
